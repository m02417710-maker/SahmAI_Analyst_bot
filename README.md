## 🎯 نبذة عن المشروع

تطبيق متكامل لتحليل الأسهم المصرية في البورصة المصرية (EGX) باستخدام الذكاء الاصطناعي والمؤشرات الفنية المتقدمة. يوفر التطبيق تحليلاً فورياً، تنبيهات آلية، وإدارة محفظة استثمارية مع تكامل كامل مع بوت تليجرام.

### ✨ المميزات الرئيسية

- 🤖 **تحليل ذكي باستخدام Google Gemini AI**
- 📊 **مؤشرات فنية متقدمة** (RSI, MACD, Bollinger Bands, المتوسطات المتحركة)
- 📈 **رسوم بيانية تفاعلية** متعددة المستويات
- 🔔 **تنبيهات فورية عبر تليجرام**
- 💼 **إدارة محفظة استثمارية** متكاملة
- ⭐ **قائمة أسهم مفضلة**
- 📱 **دعم كامل للغة العربية**
- 🌙 **وضع مظلم/فاتح**
- 📊 **مقارنة أداء الأسهم**
- 📰 **تحليل أخبار السوق**

## 🔧 التثبيت والإعداد

### 1. استنساخ المشروع

```bash
git clone https://github.com/yourusername/stock-ai-analyst.git
cd stock-ai-analyst
```

### 2. تثبيت المكتبات المطلوبة

```bash
pip install -r requirements.txt
```

محتويات ملف `requirements.txt`:

```txt
streamlit>=1.28.0
yfinance>=0.2.28
pandas>=2.0.0
numpy>=1.24.0
plotly>=5.17.0
google-generativeai>=0.3.0
pandas-ta>=0.3.14b0
openpyxl>=3.1.2
requests>=2.31.0
python-dotenv>=1.0.0
```

### 3. إعداد مفاتيح API

قم بإنشاء ملف `.streamlit/secrets.toml`:

```toml
# مفاتيح API المطلوبة
GEMINI_API_KEY = "your_gemini_api_key_here"

# (اختياري) إعدادات تليجرام
TELEGRAM_BOT_TOKEN = "your_telegram_bot_token_here"
TELEGRAM_CHAT_ID = "your_telegram_chat_id_here"
```

## 📱 استخدام التطبيق

### الصفحات الرئيسية

#### 1. 🔍 تحليل سهم واحد
- اختر السهم من القائمة المنسدلة
- عرض المؤشرات الفنية الأساسية
- رسم بياني تفاعلي مع:
  - الشموع اليابانية
  - Bollinger Bands
  - المتوسطات المتحركة (SMA 20, EMA 9)
  - مؤشر RSI
  - مؤشر MACD
  - حجم التداول
- نقاط الدعم والمقاومة
- تحليل ذكي باستخدام Gemini AI

#### 2. 📊 أداء الأسهم
- عرض جميع الأسهم مرتبة حسب الأداء
- مؤشرات السوق العامة
- أقوى الأسهم صعوداً وهبوطاً
- رسم بياني للأداء
- إحصائيات فورية

#### 3. 📰 تحليل السوق
- تحليل شامل للسوق باستخدام الذكاء الاصطناعي
- أخبار وتوقعات السوق
- تحليل القطاعات

#### 4. 📈 مقارنة الأسهم
- مقارنة أداء سهمين
- رسم بياني تفاعلي للمقارنة
- تحليل الفروقات

#### 5. ⚙️ الإعدادات
- تبديل الوضع المظلم/الفاتح
- إعدادات تليجرام
- معلومات التطبيق

### 💼 إدارة المحفظة

يمكنك إضافة صفقات الشراء والبيع:
- **شراء:** أضف سهم، عدد الأسهم، وسعر الشراء
- **بيع:** قم ببيع أسهم من محفظتك
- **عرض:** شاهد أرباحك وخسائرك
- **تقارير:** احصل على تقارير مفصلة

### 🔔 التنبيهات التلقائية

عند تفعيل تليجرام، ستصلك تنبيهات لـ:
- ✅ وصول السهم للسعر المستهدف
- 🔴 ذروة شراء (RSI > 80)
- 🟢 ذروة بيع (RSI < 20)
- 📈 اختراق المتوسطات المتحركة
- 📉 كسر المتوسطات المتحركة

## 📊 المؤشرات الفنية المستخدمة

| المؤشر | الوصف | الإشارات |
|--------|------|----------|
| **RSI** | مؤشر القوة النسبية | >70 ذروة شراء, <30 ذروة بيع |
| **MACD** | تقارب وتباعد المتوسطات | إشارات البيع والشراء |
| **Bollinger Bands** | نطاقات بولينجر | تحديد مناطق التشبع |
| **SMA 20** | متوسط متحرك بسيط 20 يوم | اتجاه السعر |
| **EMA 9** | متوسط متحرك أسي 9 يوم | إشارات سريعة |
| **Volume** | حجم التداول | تأكيد الاتجاهات |



## 🐛 حل المشكلات الشائعة

### مشكلة: "خطأ في جلب البيانات"
**الحل:** 
- تأكد من اتصال الإنترنت
- حاول تحديث الصفحة
- جرب رمز سهم مختلف مثل `COMI.CA`

### مشكلة: "GEMINI_API_KEY not found"
**الحل:**
- تأكد من وجود ملف `secrets.toml`
- تأكد من صحة المفتاح
- أعد تشغيل التطبيق

### مشكلة: بوت تليجرام لا يعمل
**الحل:**
- تأكد من صحة التوكن
- تأكد من بدء المحادثة مع البوت
- تحقق من Chat ID
